# Security Policy

## Segnalare una vulnerabilità

Se scopri una vulnerabilità di sicurezza in questo progetto, **non aprire una issue pubblica**.

Invia una segnalazione privata a: **davidemoscariello99@gmail.com**

Includi nella segnalazione:
- Descrizione della vulnerabilità
- Passi per riprodurla
- Impatto potenziale

Risponderò entro 72 ore e ti terrò aggiornato sull'avanzamento della risoluzione.

## Note sulla gestione delle credenziali

Questo progetto utilizza API key esterne. È responsabilità di chi esegue il codice:

- Non committare mai il file `.env` (è già in `.gitignore`)
- Ruotare le chiavi se si sospetta che siano state esposte
- Verificare i permessi delle proprie API key sulle piattaforme dei provider
