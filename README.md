# Web Reputational Analysis

Pipeline Python per la raccolta, normalizzazione ed esportazione dati da più fonti:

- Google Search API
- YouTube Data API
- NewsAPI
- Wikipedia
- GDELT DOC 2.0

## Obiettivo

Generare dataset strutturati (JSON/CSV) per analisi su IBM Cloud Pak for Data.

## Struttura

- collectors → raccolta dati
- processors → trasformazione e pulizia
- exporters → esportazione
- utils → funzioni comuni
- data → dati

## Stato progetto

In fase di sviluppo (documentazione completata).
